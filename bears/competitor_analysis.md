# Bears 競合分析レポート
調査日: 2026-04-14

---

## 競合一覧

### 1. Bot Express「クマアラート」（最大の競合）

- 概要: LINE公式アカウントの機能として、クマ通報・アラート配信を提供
- 導入数: 全国350以上の自治体（2025年12月時点）
- 料金: スマホ市役所導入済み自治体は追加費用なし。未導入は月額10万円〜
- 導入速度: 最短1日（ノーコード・テンプレート方式）
- 機能: 住民LINE通報（位置情報+写真）→ 職員が現地からLINE配信
- 強み: 圧倒的な導入実績数。自治体DXプラットフォーム「スマホ市役所」の一機能として展開
- 弱み: 汎用LINEツールの一機能であり、クマ対策に特化していない。鳥獣保護区の自動割り当て等の専門機能なし。大学連携なし

Bearsとの差別化ポイント:
- Bot Expressは「汎用LINE行政ツールのクマ機能」、Bearsは「クマ対策専門システム」
- Bears独自: 鳥獣保護区の自動割り当て、危険度ゾーニング、大学連携によるスペックイン
- Bot Expressの弱点: 350自治体に入っているが、ほとんどは市町村レベル。県レベルの統一システムとしての導入事例は限定的

横取り戦略:
- Bot Express導入済み自治体には「上位互換」として提案。Bot Expressの基本機能＋専門機能（鳥獣保護区・ゾーニング・大学連携）を訴求
- 県レベルで導入することで、市町村バラバラのBot Expressを統一管理できるメリットを提示
- Bot Expressは月額10万円。Bearsのスモールスタート50万円（一括）と比較し、TCOで優位性を示す

### 2. アイエスイー（ISE）「けものおと2」

- 概要: スマートフォン専用アプリ（iOS/Android）。鳥獣出没情報の共有・マップ表示
- 導入実績: 長野県（2025年9月〜）、三重県（2025年11月〜）、信濃町など
- 利用状況: 長野県で毎日4,000件アクセス、月間10万件以上
- 費用: 自治体負担（詳細未確認）
- 機能: 出没情報のマップ表示、ヒートマップ、アラート通知
- 強み: 長野県で実績あり。日経クロステックにも取り上げられ、専門性が高い
- 弱み: 専用アプリのインストールが必要（LINEに比べて普及障壁が高い）

Bearsとの差別化ポイント:
- 専用アプリ vs LINE: 高齢者・観光客の利用率で圧倒的にLINEが有利
- アプリのインストール率は低い。LINEは既に県民の8割以上がインストール済み
- けものおと2は「情報共有」が主。Bearsは「通報→管理→通知」の双方向フロー

横取り戦略:
- 長野県は「けものおと2」導入済みのため直接競合。「アプリの普及率」を数字で示す
  - 例: けものおと2のダウンロード数 vs 長野県のLINEユーザー数
- 「けものおと2と併用」という共存提案も選択肢。管理者向けにけものおと2、住民向けにBears(LINE)
- 長野以外の県で先に実績を作り、長野を最後に攻めるのが現実的

### 3. FASTBEAR（ファストベア）

- 概要: AI集約のクマ出没マップ。全国の出没情報を自動収集・可視化
- 開発: 株式会社Aisometry（東京大学・会津大学発スタートアップ）
- 公開日: 2025年12月26日
- 機能: AIエージェントが自治体公式発表・ニュースを自動収集し、地図・タイムラインで表示
- URL: kumamap.com
- 強み: 全国12.9万件超のデータを追跡。情報収集の自動化
- 弱み: 一般向け閲覧サービスであり、自治体の通報・管理システムではない

Bearsとの関係:
- 直接競合ではない。FASTBEARは「情報閲覧」、Bearsは「通報・管理・通知」
- むしろBearsのデータをFASTBEARに連携させる等の補完関係も可能

## 横取り戦略の全体方針

1. Bot Express導入済み自治体: 「県統一システム」として上位提案。市町村バラバラの運用を統合し、県全体で情報共有できる点を訴求

2. けものおと2導入済み（長野・三重）: 正面衝突を避け、他県で実績を積んでから「LINE併用」を提案。または、けものおと2との連携（管理者向け=けものおと2、住民向け=Bears）

3. 未導入自治体: Bearsの専門性（鳥獣保護区自動割当・大学連携・ゾーニング）を前面に出し、Bot Expressとの差別化で先に入る

4. 全体: 「専門特化」「大学連携」「県レベル統一」の3軸で、汎用ツール（Bot Express）や専用アプリ（けものおと2）と差別化する
