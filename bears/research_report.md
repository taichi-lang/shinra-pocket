# Bears 横展開営業リサーチレポート
対象: 東北・北海道を除くクマ出没地域（14府県）
調査日: 2026-04-14
注: 全て公開情報・実績ベース。未確認事項は明記。

---

## 北陸ブロック

### 石川県

1. 既存システム
   - 県独自のクマ通報システム: なし（電話ベースの通報が主）
   - LINE活用: 未確認（県レベルでの統一LINE通報なし）
   - 「石川県広域データ連携基盤推進事業」でクマ共生ハッカソンを実施（2025年）。データ連携・共助をテーマにしており、ITシステムへの関心が高い

2. クマ対策の現状
   - 通報フロー: 住民→警察(110)または市町村役場→県
   - 県が出前講座で住民向けクマ対策の啓発を実施
   - 計画: 石川県ツキノワグマ管理計画あり

3. 担当部署
   - 石川県 生活環境部 自然環境課
   - 連絡先: 公式サイト https://www.pref.ishikawa.lg.jp/sizen/

4. 出没情報
   - 2024年: 420件超（過去2番目の多さ）
   - 2025年春: 能美市・白山市の住宅地で目撃・人身被害
   - 2023年: 金沢市街地出没で全国ニュースに

5. 直近の事故・報道
   - 2023年 金沢市内のショッピングモール付近でクマ出没、全国報道
   - 2025年春 住宅地での人身被害

6. LINE行政サービス
   - 石川県LINE公式アカウントあり（防災等で利用実績あり）

7. 導入チャンス・障壁
   - チャンス: ハッカソン実施でIT活用への関心高い。金沢市街地出没の危機感。LINE基盤あり
   - 障壁: ハッカソンで別のソリューションが出てくる可能性

## 甲信越ブロック

### 長野県

1. 既存システム
   - 「ツキノワグマ情報マップ」: 市町村提供の目撃情報をGIS地図で公開
   - 2025年9月からリアルタイム更新のアプリ運用開始予定との報道（NBS長野放送）
   - LINE活用: 未確認（県レベル統一なし）

2. クマ対策の現状
   - 通報フロー: 住民→市町村→県
   - 「ゾーニング」によるクマと人の生活圏分けが注目（草むら刈り取り対策）
   - 2025年は被害史上最悪レベルと報道

3. 担当部署
   - 長野県 林務部 森林づくり推進課 鳥獣対策係

4. 出没情報
   - 長野市だけで2026年2月時点で255件
   - 2025年: 全国の被害者数230人超のうち長野県は上位

5. 直近の事故・報道
   - 2025年「被害史上最悪レベル」と全国報道
   - 軽井沢・小谷村等の観光地での出没が問題化

6. LINE行政サービス
   - 長野県LINE公式アカウントあり

7. 導入チャンス・障壁
   - チャンス: 被害最悪レベルで危機感最大。観光地（軽井沢等）の安全確保ニーズ。県がアプリ運用を検討中で、IT活用への姿勢がある
   - 障壁: 独自アプリ開発を進めている可能性。アプリ vs LINE のアプローチ差を説明する必要あり

## 営業優先度マトリクス

優先度の判定基準:
- A: 被害深刻 + 既存LINE通報なし + 危機感高い → 即アプローチ
- B: 出没多い + IT活用に前向き → 岐阜実績確定後にアプローチ
- C: 出没あるが件数少ない or 自前システム充実 → 種まき

| 県 | 優先度 | 理由 |
|---|---|---|
| 長野県 | A | 被害最悪レベル。アプリ検討中で「LINE vs アプリ」の提案好機 |
| 新潟県 | A | 特別警報発令の異例事態。予算確保の追い風 |
| 石川県 | A | 市街地出没で全国報道。ハッカソンでIT活用に前向き |
| 富山県 | B | 過去最多だが「クマっぷ」リニューアル済み。住民通知機能がBears提案の隙間 |
| 福井県 | B | メール配信のみ。北陸3県セット提案で効果的 |
| 群馬県 | B | 関東最多。ArcGIS導入済みだが住民LINE通報がない |
| 山梨県 | B | 富士河口湖町がLINEプッシュ実施済み。県レベル統一の提案 |
| 京都府 | B | 932件と多いが人身被害0。予防提案 |
| 兵庫県 | C | 先進県だが独自路線。連携提案で入る |
| 栃木県 | C | 市街地出没拡大中。新計画施行のタイミング |
| 滋賀県 | C | 件数少なめ。近畿モデルの一部として |
| 広島県 | C | 3県協議会あり。IT通報ニーズは未知数 |

| 鳥取県 | C | 同上 |
| 岡山県 | C | 同上 |

## 各県の導入戦略（概要）

### 長野県（最優先）
- 攻め方: 「専用アプリではなくLINEで十分」と比較提案。観光客の安全確保を切り口に
- キーメッセージ: 高齢者も使えるLINE通報 + 観光客への即時プッシュ通知
- 岐阜大学連携の実績を示し、信州大学等との連携可能性も示唆

### 新潟県（最優先）
- 攻め方: 特別警報発令の異常事態を背景に、「来シーズンまでに」のスモールスタート提案
- キーメッセージ: メール配信→LINE即時通知への進化。鳥獣被害対策支援センターとの連携
- 随意契約で今年度中に導入可能な点を強調

### 石川県（最優先）
- 攻め方: データ連携ハッカソンの文脈に乗る。「共助×データ連携」の実装版としてBears提案
- キーメッセージ: 金沢市街地出没への即時対応。県民へのワンクリック一斉通知
- 北陸3県セット「北陸モデル」への布石

### 北陸3県セット
- 攻め方: 石川を先行導入→「隣県も導入済み」で富山・福井に随意契約
- キーメッセージ: 県境を跨ぐクマの移動追跡が可能な広域連携

### 群馬県
- 攻め方: ArcGIS管理画面は充実しているが、住民側のLINE通報・プッシュ通知がない点を指摘
- キーメッセージ: 管理画面（行政側）は十分。足りないのは住民とのリアルタイム双方向通信

### 山梨県
- 攻め方: 富士河口湖町の実績を引用し、「町単位ではなく県統一で」と提案
- キーメッセージ: 観光客の安全。富士山・南アルプスという全国的な観光地
