## アプリ概要

- **アプリ名**: Shinra Pocket
- **プラットフォーム**: iOS / Android（React Native / Expo）
- **コイン**: 3種類（🔥火 / 💧水 / 🌀渦） ※相性システムなし
- **対戦方式**: CPU対戦（ふつう / つよい） + オンライン対戦
- **コンテンツ**: 6つのゲーム

## ゲーム一覧

### ゲーム1: 三目並べ（実装済みデモあり）
- **種類**: 戦略ボードゲーム
- **ボード**: 3×3マス
- **ルール**:
  - 配置フェーズ: 各プレイヤーが交互にコインを4枚ずつ配置
  - 移動フェーズ: 全8枚配置後、自分のコインを任意の空きマスへ移動（隣接制限なし）
  - 勝利条件: 縦・横・斜めに3つ揃える
  - 移動フェーズには制限時間あり（5秒→4.5→4→...最短1秒）
  - 時間切れ＝即敗北
- **AI**: ふつう（ヒューリスティック）/ つよい（Minimax + αβ枝刈り）
- **CPUコイン選択**: プレイヤーが🔥→CPUは💧 / 💧→🌀 / 🌀→🔥
- **状態**: HTMLデモ完成済み（coin-game-demo/index.html）

### ゲーム2: 一騎打ち
- **種類**: 戦略ボードゲーム（重ね系）
- **ボード**: 3x3マス
- **ルール**:
  - 各プレイヤーはコイン4枚（数字「1」x2、数字「2」x2）を持つ
  - 毎ターン「配置」か「移動」のいずれか1つを行う
  - 配置: 手持ちコインを空きマスに置く。または「1」の上に「2」を重ねられる（敵味方問わず）
  - 移動: ボード上の自分の最上面コインを空きマスへ移動（移動での重ねは不可）
  - お手付きルール: コインに触れたらキャンセル不可、必ず動かす
  - 勝利条件: 各マスの最上面コインで縦・横・斜めに3つ揃える
  - 上のコインを動かして下のコインが露出→それで3つ揃えば、その所有者の勝ち
- **AI仕様**:
  - ふつう: ヒューリスティック（勝てるなら勝つ→ブロック→戦略配置）
  - つよい: ミニマックス系AI
- **CPUコイン選択**: Game1と同じ法則（火→水、水→渦、渦→火）
- **状態**: 実装済み

### ゲーム3: 三つ巴（一騎打ち 3人版）
- **種類**: 3人対戦戦略ボードゲーム（重ね系）
- **ボード**: 3x3マス
- **ルール**:
  - 3プレイヤー（火/水/渦）が各6枚のコイン（数字「1」x2、「2」x2、「3」x2）を持つ
  - ターン順: 火→水→渦→火→...のループ
  - 毎ターン「配置」か「移動」のいずれか1つを行う
  - 配置: 空きマスに置く、または小さい数字の上に大きい数字を重ねられる（最大3層）
  - 移動: 自分の最上面コインを空きマスへ移動
  - お手付きルール: コインに触れたらキャンセル不可
  - 勝利条件: 各マスの最上面コインで縦・横・斜めに3つ揃える（最初に揃えた1人が勝ち、残り2人敗北）
  - 漁夫の利が発生しうる
- **AI仕様**:
  - ふつう: ヒューリスティック（勝てるなら勝つ→妨害→重ねで封じ→ランダム）
  - つよい: Max^N アルゴリズム（3プレイヤー版ミニマックス）
  - CPU同士は結託せず独立して最善手を選ぶ
- **プレイモード**: CPU対戦（vs CPU2体）/ ローカル3人対戦
- **状態**: 実装済み

### ゲーム4: パタパタ
- **種類**: マンカラ系ボードゲーム
- **ボード**: 各プレイヤー3マス + 共有ピット2つ（左右）
- **ルール**:
  - 初期配置: 各プレイヤーの自陣3マスに左から4枚・3枚・2枚（合計18枚）
  - 自陣のマスを1つ選び、全コインを取り上げて反時計回りに1枚ずつ配る
  - ピット（左右どちらでも）もスキップせず配る
  - 最後の1枚がピットに落ちた場合→もう一度自分のターン（連続ターン）
  - 最後の1枚がピット以外→相手のターン
  - 横取りなし（シンプル配りのみ）
  - 勝利条件: 自陣3マスのコインを先に全て0枚にした方が勝ち
- **AI仕様**:
  - ふつう: ピットに入るなら優先→多く配れる手→ランダム
  - つよい: ミニマックス（連続ターン評価高、自陣残コイン最小化）
- **プレイモード**: CPU対戦 / ローカル対戦 / オンライン対戦
- **状態**: 実装済み

### ゲーム5: 日月の戦い
- **種類**: 簡易将棋（3x3）
- **ボード**: 3x3マス
- **ルール**:
  - 2陣営: ☀️お日様 vs 🌙月。各陣営に王（日/月）+ 火 + 水の3駒
  - 駒の動き: 王=全8方向1マス、火=斜め4方向1マス、水=前後2方向1マス
  - 初期配置: 自陣1列目に火・王・水（左から）
  - 毎ターン「駒を動かす」か「持ち駒を打つ」のいずれか1つ
  - 移動先に相手の駒があれば捕獲→自分の駒として手持ちに
  - 持ち駒は自陣1列目の空きマスにのみ打てる
  - 勝利条件: 相手の王を詰ませる（王手を解除できない状態）
  - 引き分け: 同一局面3回繰り返し（千日手）
- **AI仕様**:
  - ふつう: ヒューリスティック（取れるなら取る→王手を狙う→ランダム）
  - つよい: ミニマックス＋αβ枝刈り（3x3+持ち駒で完全解析も視野）
- **プレイモード**: CPU対戦 / ローカル対戦 / オンライン対戦
- **状態**: 実装済み

### ゲーム6: Number Link（3x3 Quiz）
- **種類**: 数字パズル（1人プレイ）
- **ボード**: 3x3マス
- **ルール**:
  - 一部がヒントとして埋まった3x3グリッドの空セルに数字を入れる
  - 各行の合計と各列の合計が表示されたターゲット数になれば正解
  - 難易度3段階: Easy(数字1-3,目標6,制限60秒) / Normal(1-5,目標9,45秒) / Hard(1-9,目標15,30秒)
  - パズルモード: 順次出題、3ライフ制（不正解で1ライフ減、0でゲームオーバー）
  - タイムアタックモード: 制限時間内に何問解けるか（正解でボーナス時間追加）
  - スコアリング: Easy100pt / Normal200pt / Hard400pt + 残り時間ボーナス + コンボ(最大x3)
  - ヒント機能: 1パズルにつき最大2回、空セルの正解を1つ表示（-50pt）
- **AI仕様**: なし（1人プレイのためCPU対戦なし。パズル自動生成アルゴリズムで出題）
- **プレイモード**: パズルモード / タイムアタックモード
- **状態**: 実装済み

### 画面構成
