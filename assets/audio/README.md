# Audio Assets for ShinraPocket

Place audio files in the directories below. The game handles missing files gracefully (no crash).

## SFX (assets/audio/sfx/)

| File                | Description                        |
|---------------------|------------------------------------|
| coin_place.mp3      | Coin placed on board               |
| coin_move.mp3       | Coin moved on board                |
| victory.mp3         | Game victory fanfare               |
| defeat.mp3          | Game defeat sound                  |
| timer_warning.mp3   | Timer countdown warning (last 5s)  |
| ui_tap.mp3          | UI button tap                      |
| ui_back.mp3         | UI back / cancel                   |
| match_found.mp3     | Online match found                 |
| game_start.mp3      | Game start signal                  |
| turn_change.mp3     | Turn change notification           |
| combo.mp3           | Combo / chain reaction             |
| error.mp3           | Invalid action / error             |
| reward.mp3          | Reward received                    |
| level_up.mp3        | Level up                           |
| countdown.mp3       | 3-2-1 countdown beep              |

## BGM (assets/audio/bgm/)

| File                  | Description              |
|-----------------------|--------------------------|
| lobby.mp3             | Main menu / lobby music  |
| game1_coinboard.mp3   | Coin Board game BGM      |
| game2.mp3             | Game 2 BGM               |
| game3.mp3             | Game 3 BGM               |
| game4.mp3             | Game 4 BGM               |
| game5.mp3             | Game 5 BGM               |
| game6_quiz.mp3        | 3x3 Quiz game BGM       |
| battle.mp3            | Online battle BGM        |
| results.mp3           | Results screen BGM       |

## Format Requirements

- Format: MP3 (recommended) or WAV
- Sample rate: 44100 Hz
- SFX: short clips (< 3 seconds recommended)
- BGM: loopable tracks
