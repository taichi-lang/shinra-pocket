# SNS運用エージェントチーム

なづInstagram運用のためのAIエージェント群。
候補Cの各エージェント（Content/Monitor/Support/Ops/Growth）の原型。

## チーム構成

### 運用チーム（日常運用）
| エージェント | 役割 | 頻度 |
|---|---|---|
| **Content** | 写真選定・キャプション生成・投稿準備 | 週2回 |
| **Monitor** | インサイト分析・レポート生成 | 週1回 |
| **Support** | コメント・DM返信ドラフト作成 | 毎日（通知時） |
| **Ops** | スケジュール管理・投稿キュー・品質チェック | 常時 |
| **Growth** | フォロワー増加施策・トレンド分析 | 週1回 |

### 営業チーム（M6〜）
| エージェント | 役割 | 頻度 |
|---|---|---|
| **Scout** | PR案件候補のリサーチ・リスト作成 | 月2回 |
| **Outreach** | 営業メール・DM文面の生成 | 案件ごと |
| **Deal** | 案件条件の整理・交渉ポイント提示 | 案件ごと |

## データフロー

```
写真フォルダ (D:\保存\Backup\)
    ↓
[Content] 写真選定 → キャプション生成
    ↓
[Ops] 品質チェック → 投稿キューに追加
    ↓
太一さん承認
    ↓
Instagram投稿
    ↓
[Monitor] インサイト取得 → レポート
    ↓
[Support] コメント対応
    ↓
[Growth] 改善提案 → Content にフィードバック
```

## 営業フロー（M6〜）

```
[Scout] 候補企業リサーチ
    ↓
[Outreach] 営業文面生成
    ↓
太一さん承認 → 送信
    ↓
[Deal] 条件整理・交渉サポート
```
